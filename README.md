## Installation

`$ npm install -g console-brain-games`

## Game Start

### Brain Even

`$ brain-even`

### Brain Calc

`$ brain-calc`

### Brain Gcd

`$ brain-gcd`

### Brain Progression

`$ brain-progression`

### Brain Prime

`$ brain-prime`

## Uninstall

`$ npm uninstall -g console-brain-games`
